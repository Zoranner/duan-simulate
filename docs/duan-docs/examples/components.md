# 示例组件

> **重要说明**：本文档中的组件仅为示例，用于展示如何使用 DUAN 框架设计组件。这些组件不是框架的一部分，用户应根据具体仿真场景自行定义和实现组件。

本文档提供几个常见组件的参考设计，帮助理解组件的设计思路。

## 位置组件

### 描述

描述实体在空间中的位置和朝向。

### 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 位置 | 三维向量 | 实体的空间坐标 |
| 朝向 | 四元数 | 实体的旋转方向 |

### 使用场景

- 空间域：距离计算、视线检测
- 运动域：位置更新
- 探测域：距离判断
- 战斗域：命中判定

### 注意事项

- 位置通常使用三维坐标（x, y, z）
- 朝向可以使用欧拉角或四元数表示
- 坐标系需要统一（如世界坐标或局部坐标）

## 运动组件

### 描述

描述实体的运动参数。

### 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 速度 | 三维向量 | 当前速度向量 |
| 最大速度 | 数值 | 最大速度限制 |
| 转向率 | 数值 | 最大转向角速度 |
| 目标位置 | 三维向量（可选） | 目标位置（用于路径跟踪） |

### 使用场景

- 运动域：运动计算、位置更新

### 注意事项

- 速度向量包含方向和大小
- 转向率影响实体的机动性
- 目标位置用于自动导航

## 雷达组件

### 描述

描述实体的雷达配置和能力。

### 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 雷达类型 | 字符串 | 雷达的类型标识 |
| 探测范围 | 数值 | 最大探测距离 |
| 扫描间隔 | 数值 | 扫描周期 |
| 方向性 | 数值（可选） | 扫描角度范围 |

### 使用场景

- 探测域：探测能力判断、探测范围确定

### 注意事项

- 雷达类型可用于区分不同性能的雷达
- 探测范围影响能探测到的目标距离
- 扫描间隔影响探测的实时性

## 武器组件

### 描述

描述实体的武器配置和能力。

### 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 武器类型 | 字符串 | 武器的类型标识 |
| 射程 | 数值 | 最大射程 |
| 射速 | 数值 | 每秒发射次数 |
| 弹药 | 数值 | 剩余弹药数量 |
| 伤害 | 数值 | 每次命中的伤害值 |

### 使用场景

- 战斗域：攻击能力判断、伤害计算

### 注意事项

- 武器类型可用于区分不同类型的武器
- 弹药数量可能影响攻击能力
- 射速和伤害决定输出能力

## 阵营组件

### 描述

描述实体的阵营归属。

### 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 阵营标识 | 整数或字符串 | 阵营的唯一标识 |

### 使用场景

- 阵营域：敌我关系判断
- 探测域：目标筛选
- 战斗域：攻击目标选择

### 注意事项

- 阵营标识需要与阵营域的配置匹配
- 常见的阵营：红方、蓝方、中立

## 阵型组件

### 描述

描述实体在编队中的位置。

### 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 阵型类型 | 字符串 | 阵型的类型标识 |
| 偏移位置 | 三维向量 | 相对于编队中心的位置 |
| 编队标识 | 实体标识（可选） | 所属编队的实体标识 |

### 使用场景

- 阵型域：编队管理、阵型位置计算

### 注意事项

- 偏移位置用于计算实体在阵型中的位置
- 编队标识指向编队的领导实体

## 定时器组件

### 描述

描述实体的定时器列表。

### 字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 定时器列表 | 定时器数组 | 实体的所有定时器 |

### 定时器字段

| 字段 | 类型 | 说明 |
|------|------|------|
| 标识 | 字符串 | 定时器的唯一标识 |
| 触发时间 | 数值 | 触发的仿真时间点 |
| 是否循环 | 布尔 | 是否周期性触发 |
| 循环间隔 | 数值（可选） | 循环的间隔时间 |
| 回调类型 | 枚举 | 触发时执行的操作 |

### 使用场景

- 时间管理：定时触发事件

### 注意事项

- 触发时间使用仿真时间
- 回调类型可以是自毁、发送事件、自定义回调等

## 组件设计模式

### 属性组件模式

描述实体的基本属性，字段简单，广泛使用。

示例：位置组件、阵营组件。

### 配置组件模式

描述实体的配置参数，创建时设置，运行时通常不变。

示例：雷达组件、武器组件。

### 状态组件模式

描述实体的运行时状态，由域在仿真过程中更新。

示例：运动组件（速度可能变化）。

### 关系组件模式

描述实体与其他实体的关系。

示例：阵型组件（包含编队标识）。

## 组件字段类型选择

### 数值类型

- **整数**：用于计数、标识
- **浮点数**：用于位置、速度、范围等连续量
- **向量**：用于位置、速度等有方向的量
- **四元数**：用于旋转

### 字符串类型

- 用于类型标识、名称
- 注意：避免过度使用，可能影响性能

### 布尔类型

- 用于开关、状态标志

### 枚举类型

- 用于有限选项的字段
- 注意：考虑可扩展性

### 集合类型

- 用于多个值的存储
- 注意：考虑容量和性能

### 可选类型

- 用于可能缺失的字段
- 注意：处理缺失情况
